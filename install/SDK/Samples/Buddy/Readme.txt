Buddy.dll

This is a sample to move 3D objects .


To install, do the following:

(1) Add the Buddy.dll file to the \FlyLegacy\Modules\PC directory.

(2) Add sobj_cessna.sit to the FlyLegacy "WorldObj" folder

(3) Add cessna.lod and cessna.nfo to the FlyLegacy "World" folder

In the Deauville (LFRG.sit) airport you should see two cessna moving around.
